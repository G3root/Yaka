# yaka

Terminal UI for managing projects, deployments, domains, servers, and secrets from the terminal.

## Install

With Bun:

```bash
bun install
```

Run locally:

```bash
bun run dev
```

Build the CLI bundle:

```bash
bun run build:cli
```

Build the standalone release binary:

```bash
bun run build:standalone
```

Build npm packages:

```bash
bun run build:npm-packages
```
